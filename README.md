# Анализатор банковских транзакций

Проект для анализа банковских операций из Excel-файла.

## Возможности

### Сервисы

- Анализ выгодных категорий кешбэка
- Расчёт суммы для Инвесткопилки
- Простой поиск по транзакциям
- Поиск по телефонным номерам
- Поиск переводов физическим лицам

### Веб-страницы

- Главная страница
- Страница событий

### Отчёты

- Траты по категории
- Траты по дням недели
- Траты по рабочим и выходным дням
