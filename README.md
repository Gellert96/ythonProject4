# Анализатор банковских транзакций

Проект для анализа банковских операций из Excel-файла с помощью Python.

Приложение позволяет анализировать транзакции, выполнять поиск по операциям, рассчитывать кешбэк, формировать данные для веб-страниц и строить отчёты.

## Функциональность

### Сервисы

- Анализ выгодных категорий повышенного кешбэка
- Расчёт суммы накоплений для «Инвесткопилки»
- Простой поиск по транзакциям
- Поиск транзакций по номеру телефона
- Поиск переводов физическим лицам

### Веб-страницы

- Главная страница:
  - приветствие по времени суток
  - информация по картам
  - топ-5 транзакций
  - курсы валют
  - стоимость акций

- Страница событий:
  - расходы по категориям
  - поступления
  - курсы валют
  - стоимость акций

### Отчёты

- Траты по категории
- Траты по дням недели
- Траты в рабочие и выходные дни

## Установка

Клонировать репозиторий:

```bash
git clone <repository_url>
```

Перейти в папку проекта:

```bash
cd PythonProject4
```

Установить зависимости:

```bash
poetry install
```

Активировать виртуальное окружение:

```bash
poetry shell
```

---

## Запуск проекта

Запуск приложения:

```bash
python -m src.main
```

После запуска доступно меню выбора сервиса:

1. Выгодные категории кешбэка
2. Инвесткопилка
3. Простой поиск
4. Поиск по телефонам
5. Поиск переводов физическим лицам
